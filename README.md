# JavaScript Learning Project

A comprehensive collection of JavaScript applications and learning exercises showcasing various web development concepts and APIs.

## 🚀 Features

This project includes several interactive web applications:

### 🌤️ Weather App
- Real-time weather data using OpenWeatherMap API
- City-based weather lookup
- Temperature, humidity, and weather conditions display
- Dynamic weather emojis

### 🧮 Calculator
- Basic arithmetic operations
- Clean, intuitive interface
- Real-time calculation display

### ⏱️ Stopwatch
- Start, stop, and reset functionality
- Precise time tracking
- Clean digital display

### 🎮 Pokémon Lookup
- Fetch Pokémon data from PokéAPI
- Display Pokémon sprites
- Search by Pokémon name

### 📚 Learning Examples
- DOM manipulation examples
- Event handling demonstrations
- JavaScript fundamentals practice
- HTML forms and validation
- CSS styling experiments

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **APIs**: OpenWeatherMap API, PokéAPI
- **Tools**: Fetch API, DOM manipulation, Event listeners

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open any HTML file** in your web browser
3. **For weather app**: Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
4. **Start exploring** the different applications!

## 💡 Usage Examples

### Weather App
1. Open `weather.html`
2. Enter a city name
3. Click "Get Weather" to see current conditions

### Calculator
1. Open `calculator.html`
2. Click buttons to perform calculations
3. Use the display to see results

### Stopwatch
1. Open `stopwatch.html`
2. Click "Start" to begin timing
3. Use "Stop" and "Reset" as needed
